## 🛠️ Useful Tools & Utilities

### Password Reset
```bash
cd backend
node reset-password.mjs <email> <new-password>
```

### Run Tests
```bash
cd backend
node test-auth-local.mjs
node test-api-e2e-local.mjs
```

### Check Database
```bash
cd backend
PGPASSWORD=*** psql -h 127.0.0.1 -U chama_app -d chamaPlus
```

### Deploy
```bash
cd backend
./deploy.sh
```
